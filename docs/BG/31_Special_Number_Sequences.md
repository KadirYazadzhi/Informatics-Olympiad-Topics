## 1. –ß–∏—Å–ª–∞ –Ω–∞ –ö–∞—Ç–∞–ª–∞–Ω (Catalan Numbers)

–ü–æ—Ä–µ–¥–∏—Ü–∞—Ç–∞ –∑–∞–ø–æ—á–≤–∞ —Ç–∞–∫–∞: $1, 1, 2, 5, 14, 42, 132, 429, 1430, \dots$ –∑–∞ $n = 0, 1, 2, \dots$.

### 1.1. –§–æ—Ä–º—É–ª–∏
–ù–∞–π-–ø–æ–ø—É–ª—è—Ä–Ω–∞—Ç–∞ —Ñ–æ—Ä–º—É–ª–∞ –µ:
$$C_n = \frac{1}{n+1} \binom{2n}{n} = \frac{(2n)!}{(n+1)!n!}$$ 

–†–µ–∫—É—Ä–µ–Ω—Ç–Ω–∞—Ç–∞ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç, –∫–æ—è—Ç–æ —á–µ—Å—Ç–æ —Å–µ –ø–æ–ª–∑–≤–∞ –≤ –î–ü:
$$C_{n+1} = \sum_{i=0}^{n} C_i C_{n-i}$$ 

### 1.2. –ö–ª–∞—Å–∏—á–µ—Å–∫–∏ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è

## 2. –ß–∏—Å–ª–∞ –Ω–∞ –§–∏–±–æ–Ω–∞—á–∏ (Fibonacci)

–î–µ—Ñ–∏–Ω–∏—Ä–∞–Ω–∏ —á—Ä–µ–∑: $F_0 = 0, F_1 = 1, F_n = F_{n-1} + F_{n-2}$.

### 2.1. –ë—ä—Ä–∑–æ –ø—Ä–µ—Å–º—è—Ç–∞–Ω–µ ($O(\log N)$)
–ó–∞ –º–Ω–æ–≥–æ –≥–æ–ª–µ–º–∏ $N$ ($10^{18}$) –∏–∑–ø–æ–ª–∑–≤–∞–º–µ –º–∞—Ç—Ä–∏—á–Ω–æ –ø–æ–≤–¥–∏–≥–∞–Ω–µ –Ω–∞ —Å—Ç–µ–ø–µ–Ω:
$$ \begin{pmatrix} F_{n+1} \\ F_n \end{pmatrix} = \begin{pmatrix} 1 & 1 \\ 1 & 0 \end{pmatrix}^n \begin{pmatrix} 1 \\ 0 \end{pmatrix} $$ 

### 2.2. –°–≤–æ–π—Å—Ç–≤–∞
1.  **–°—É–º–∞**: $\sum_{i=1}^n F_i = F_{n+2} - 1$.
2.  **–ü–µ—Ä–∏–æ–¥ –Ω–∞ –ü–∏–∑–∞–Ω–æ**: –û—Å—Ç–∞—Ç—ä—Ü–∏—Ç–µ –Ω–∞ –§–∏–±–æ–Ω–∞—á–∏ –ø–æ –º–æ–¥—É–ª $M$ –æ–±—Ä–∞–∑—É–≤–∞—Ç —Ü–∏–∫—ä–ª.
3.  **–ó–ª–∞—Ç–Ω–æ —Å–µ—á–µ–Ω–∏–µ**: –ì—Ä–∞–Ω–∏—Ü–∞—Ç–∞ –Ω–∞ $F_{n+1}/F_n$ –µ $\phi \approx 1.618$.
4.  **–§–æ—Ä–º—É–ª–∞ –Ω–∞ –ë–∏–Ω–µ**: $F_n = \frac{\phi^n - \psi^n}{\sqrt{5}}$ –∫—ä–¥–µ—Ç–æ $\phi = \frac{1+\sqrt{5}}{2}$ –∏ $\psi = \frac{1-\sqrt{5}}{2}$.
5.  **GCD —Å–≤–æ–π—Å—Ç–≤–æ**: $\gcd(F_m, F_n) = F_{\gcd(m,n)}$.

### 2.3. –ò–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è –Ω–∞ –º–∞—Ç—Ä–∏—á–Ω–æ —Å—Ç–µ–ø–µ–Ω—É–≤–∞–Ω–µ

        for (int i = 0; i < 2; i++) {

                for (int k = 0; k < 2; k++) {
                    res.a[i][j] += a[i][k] * other.a[k][j];
                    res.a[i][j] %= MOD;
                }
            }
        }
        return res;
    }
};

Matrix matpow(Matrix base, long long exp) {
    Matrix res;
    res.a[0][0] = res.a[1][1] = 1;  // Identity matrix

        if (exp % 2) res = res * base;
        base = base * base;

}

    if (n <= 1) return n;
    
    Matrix base;
    base.a[0][0] = base.a[0][1] = base.a[1][0] = 1;
    base.a[1][1] = 0;

}

    vector<int> fibs;
    fibs.push_back(1);
    fibs.push_back(2);
    
    while (fibs.back() < n) {
        int sz = fibs.size();
        fibs.push_back(fibs[sz-1] + fibs[sz-2]);
    }

## 3. –ß–∏—Å–ª–∞ –Ω–∞ –°—Ç–∏—Ä–ª–∏–Ω–≥ (Stirling Numbers)

### 3.1. –í—Ç–æ—Ä–∏ —Ä–æ–¥ $S(n, k)$
–ë—Ä–æ—è—Ç –Ω–∞—á–∏–Ω–∏ –¥–∞ —Å–µ —Ä–∞–∑–±–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –æ—Ç $n$ –µ–ª–µ–º–µ–Ω—Ç–∞ –Ω–∞ $k$ **–Ω–µ–ø—Ä–∞–∑–Ω–∏** –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞.
*   **–†–µ–∫—É—Ä—Å–∏—è**: $S(n, k) = k \cdot S(n-1, k) + S(n-1, k-1)$.
    *   *–û–±—è—Å–Ω–µ–Ω–∏–µ*: –ù–æ–≤–∏—è—Ç –µ–ª–µ–º–µ–Ω—Ç –∏–ª–∏ –≤–ª–∏–∑–∞ –≤ –µ–¥–Ω–æ –æ—Ç —Å—ä—â–µ—Å—Ç–≤—É–≤–∞—â–∏—Ç–µ $k$ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞, –∏–ª–∏ —Å—ä–∑–¥–∞–≤–∞ –Ω–æ–≤–æ —Å–∞–º.

### 3.2. –ü—ä—Ä–≤–∏ —Ä–æ–¥ $c(n, k)$
–ë—Ä–æ—è—Ç –Ω–∞ –ø–µ—Ä–º—É—Ç–∞—Ü–∏–∏—Ç–µ –Ω–∞ $n$ –µ–ª–µ–º–µ–Ω—Ç–∞, –∫–æ–∏—Ç–æ —Å–µ —Å—ä—Å—Ç–æ—è—Ç –æ—Ç —Ç–æ—á–Ω–æ $k$ —Ü–∏–∫—ä–ª–∞.
*   **–†–µ–∫—É—Ä—Å–∏—è**: $c(n, k) = (n-1)c(n-1, k) + c(n-1, k-1)$.

### 3.3. –ò–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è

## 4. –ß–∏—Å–ª–∞ –Ω–∞ –ë–µ–ª (Bell Numbers) $B_n$

–û–±—â–∏—è—Ç –±—Ä–æ–π –Ω–∞ –≤—Å–∏—á–∫–∏ –≤—ä–∑–º–æ–∂–Ω–∏ —Ä–∞–∑–±–∏–≤–∞–Ω–∏—è –Ω–∞ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –æ—Ç $n$ –µ–ª–µ–º–µ–Ω—Ç–∞ (–Ω–∞ –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω –±—Ä–æ–π –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞).
$$B_n = \sum_{k=0}^n S(n, k)$$
–†–µ–∫—É—Ä—Å–∏—è: $B_{n+1} = \sum_{k=0}^n \binom{n}{k} B_k$.

### 4.1. –ò–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è

## 2. –°–∏–ª–Ω–æ –°–≤—ä—Ä–∑–∞–Ω–∏ –ö–æ–º–ø–æ–Ω–µ–Ω—Ç–∏ (SCC)

–í –Ω–∞—Å–æ—á–µ–Ω –≥—Ä–∞—Ñ, SCC (Strongly Connected Component) –µ –º–∞–∫—Å–∏–º–∞–ª–Ω–æ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ –æ—Ç –≤—ä—Ä—Ö–æ–≤–µ, –∫—ä–¥–µ—Ç–æ –≤—Å–µ–∫–∏ –≤—Ä—ä—Ö –µ –¥–æ—Å—Ç–∏–∂–∏–º –æ—Ç –≤—Å–µ–∫–∏ –¥—Ä—É–≥ –≤—Ä—ä—Ö –≤ —Ç–æ–≤–∞ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ. –° –¥—Ä—É–≥–∏ –¥—É–º–∏, –∑–∞ –≤—Å–µ–∫–∏ –¥–≤–∞ –≤—ä—Ä—Ö–∞ $u$ –∏ $v$ –≤ –µ–¥–∏–Ω SCC, —Å—ä—â–µ—Å—Ç–≤—É–≤–∞ –ø—ä—Ç –æ—Ç $u$ –¥–æ $v$ –ò –ø—ä—Ç –æ—Ç $v$ –¥–æ $u$.

**–í–∞–∂–Ω–æ —Å–≤–æ–π—Å—Ç–≤–æ:** –ö–æ–Ω–¥–µ–Ω–∑–∞—Ü–∏—è—Ç–∞ –Ω–∞ –≥—Ä–∞—Ñ–∞ (—Å–≤–∏–≤–∞–Ω–µ—Ç–æ –Ω–∞ –≤—Å–µ–∫–∏ SCC –≤ –µ–¥–∏–Ω —Å—É–ø–µ—Ä-–≤—Ä—ä—Ö) –≤–∏–Ω–∞–≥–∏ –¥–∞–≤–∞ DAG (Directed Acyclic Graph). –¢–æ–≤–∞ —Å–≤–æ–π—Å—Ç–≤–æ –µ –∏–∑–∫–ª—é—á–∏—Ç–µ–ª–Ω–æ –ø–æ–ª–µ–∑–Ω–æ –∑–∞ —Ä–µ—à–∞–≤–∞–Ω–µ –Ω–∞ –º–Ω–æ–≥–æ –∑–∞–¥–∞—á–∏.

### 2.1. –ê–ª–≥–æ—Ä–∏—Ç—ä–º –Ω–∞ –ö–æ—Å–∞—Ä–∞–π—é (Kosaraju's Algorithm)

–¢–æ–∑–∏ –∞–ª–≥–æ—Ä–∏—Ç—ä–º –µ –ø–æ-–∏–Ω—Ç—É–∏—Ç–∏–≤–µ–Ω –∏ –∏–∑–ø–æ–ª–∑–≤–∞ –¥–≤–∞ DFS –æ–±—Ö–æ–∂–¥–∞–Ω–∏—è:

```cpp

int comp[MAXN];     // –ù–æ–º–µ—Ä –Ω–∞ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –∑–∞ –≤—Å–µ–∫–∏ –≤—Ä—ä—Ö

void dfs1(int u) {
    visited[u] = true;
    for (int v : adj[u]) {
        if (!visited[v]) dfs1(v);
    }
    order.push_back(u);  // –î–æ–±–∞–≤—è–º–µ –ø—Ä–∏ –∑–∞–≤—ä—Ä—à–≤–∞–Ω–µ
}

void dfs2(int u, int component) {
    comp[u] = component;
    for (int v : adj_rev[u]) {
        if (comp[v] == -1) dfs2(v, component);
    }
}

void findSCC_Kosaraju(int n) {
    // –°—Ç—ä–ø–∫–∞ 1: –ü—ä—Ä–≤–∏ DFS –∑–∞ –Ω–∞–º–∏—Ä–∞–Ω–µ –Ω–∞ —Ä–µ–¥–∞ –Ω–∞ –ø—Ä–∏–∫–ª—é—á–≤–∞–Ω–µ
    for (int i = 1; i <= n; i++) {
        if (!visited[i]) dfs1(i);
    }
    
    // –°—Ç—ä–ø–∫–∞ 2: DFS –Ω–∞ –æ–±—ä—Ä–Ω–∞—Ç–∏—è –≥—Ä–∞—Ñ –≤ –æ–±—Ä–∞—Ç–µ–Ω —Ä–µ–¥
    fill(comp, comp + n + 1, -1);
    int num_components = 0;
    reverse(order.begin(), order.end());
    
    for (int u : order) {
        if (comp[u] == -1) {
            dfs2(u, num_components++);
        }
    }
}
```

### 2.2. –ê–ª–≥–æ—Ä–∏—Ç—ä–º –Ω–∞ –¢–∞—Ä—è–Ω –∑–∞ SCC

–ò–∑–ø–æ–ª–∑–≤–∞ –µ–¥–Ω–æ DFS –æ–±—Ö–æ–∂–¥–∞–Ω–µ –∏ —Å—Ç–µ–∫. –ü–æ-–µ—Ñ–µ–∫—Ç–∏–≤–µ–Ω –æ—Ç –ö–æ—Å–∞—Ä–∞–π—é:

```cpp
vector<int> adj[MAXN];
int tin[MAXN], low[MAXN], timer = 0;
bool on_stack[MAXN];
stack<int> st;
int comp[MAXN], num_components = 0;

void dfs(int u) {
    tin[u] = low[u] = timer++;
    st.push(u);
    on_stack[u] = true;
    
    for (int v : adj[u]) {
        if (tin[v] == -1) {
            // v –Ω–µ –µ –ø–æ—Å–µ—Ç–µ–Ω
            dfs(v);
            low[u] = min(low[u], low[v]);
        } else if (on_stack[v]) {
            // v –µ –≤ —Ç–µ–∫—É—â–∏—è SCC
            low[u] = min(low[u], tin[v]);
        }
    }
    
    // –ê–∫–æ u –µ –∫–æ—Ä–µ–Ω—ä—Ç –Ω–∞ SCC
    if (low[u] == tin[u]) {
        while (true) {
            int v = st.top();
            st.pop();
            on_stack[v] = false;
            comp[v] = num_components;
            if (v == u) break;
        }
        num_components++;
    }
}

void findSCC_Tarjan(int n) {
    fill(tin, tin + n + 1, -1);
    fill(on_stack, on_stack + n + 1, false);
    num_components = 0;
    
    for (int i = 1; i <= n; i++) {
        if (tin[i] == -1) {
            dfs(i);
        }
    }
}
```

## 3. 2-SAT (2-Satisfiability)

### 3.1. –°–≤–µ–∂–¥–∞–Ω–µ –¥–æ SCC

–£—Å–ª–æ–≤–∏–µ—Ç–æ $(a \lor b)$ –µ –µ–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ –Ω–∞ –¥–≤–µ –∏–º–ø–ª–∏–∫–∞—Ü–∏–∏:

**–ê–ª–≥–æ—Ä–∏—Ç—ä–º:**

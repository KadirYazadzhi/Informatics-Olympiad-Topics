# Informatics Olympiad Topics

This repository hosts the source code for the [Informatics Olympiad Topics website](https://anonymous.github.io/Informatics-Olympiad-Topics/).

## Website Structure

The content is organized in the `docs/` directory:
- `docs/EN/`: English topics
- `docs/BG/`: Bulgarian topics

## Building Locally

To preview the site locally:

1. Install MkDocs and the Material theme:
   ```bash
   pip install mkdocs-material
   ```
2. Run the development server:
   ```bash
   mkdocs serve
   ```
3. Open `http://127.0.0.1:8000` in your browser.

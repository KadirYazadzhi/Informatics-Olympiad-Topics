## 2. –ù–∞–π-–≥–æ–ª—è–º –æ–±—â –¥–µ–ª–∏—Ç–µ–ª (–ù–û–î / GCD)

–ù–∞–π-–≥–æ–ª—è–º–æ—Ç–æ —á–∏—Å–ª–æ, –∫–æ–µ—Ç–æ –¥–µ–ª–∏ –±–µ–∑ –æ—Å—Ç–∞—Ç—ä–∫ –µ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ $A$ –∏ $B$.

### 2.1. –ê–ª–≥–æ—Ä–∏—Ç—ä–º –Ω–∞ –ï–≤–∫–ª–∏–¥
–¢–æ–≤–∞ –µ –µ–¥–∏–Ω –æ—Ç –Ω–∞–π-—Å—Ç–∞—Ä–∏—Ç–µ –∏ –≤–∞–∂–Ω–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∏. –ë–∞–∑–∏—Ä–∞ —Å–µ –Ω–∞ —Ñ–∞–∫—Ç–∞, —á–µ $\gcd(a, b) = \gcd(b, a \pmod b)$.

```cpp
// –†–µ–∫—É—Ä—Å–∏–≤–µ–Ω –≤–∞—Ä–∏–∞–Ω—Ç
int gcd(int a, int b) {
    if (b == 0) return a;
    return gcd(b, a % b);
}

// –ò—Ç–µ—Ä–∞—Ç–∏–≤–µ–Ω –≤–∞—Ä–∏–∞–Ω—Ç
int gcd_iter(int a, int b) {
    while (b) {
        a %= b;
        swap(a, b);
    }
    return a;
}
```

*   **–í–≥—Ä–∞–¥–µ–Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è**: –í C++ (`<algorithm>` –∏–ª–∏ `<numeric>`) –∏–º–∞ —Ñ—É–Ω–∫—Ü–∏—è `std::gcd(a, b)` (–æ—Ç C++17) –∏–ª–∏ `__gcd(a, b)` (GCC built-in, —Ä–∞–±–æ—Ç–∏ –∏ –≤ –ø–æ-—Å—Ç–∞—Ä–∏ –≤–µ—Ä—Å–∏–∏).

### 2.2. –°–≤–æ–π—Å—Ç–≤–∞ –Ω–∞ –ù–û–î
*   $\\gcd(a, b) = \gcd(a, b-a)$
*   $\\gcd(k \cdot a, k \cdot b) = k \cdot \gcd(a, b)$
*   $\\gcd(a, b, c) = \gcd(a, \gcd(b, c))$

## 3. –ù–∞–π-–º–∞–ª–∫–æ –æ–±—â–æ –∫—Ä–∞—Ç–Ω–æ (–ù–û–ö / LCM)

–ù–∞–π-–º–∞–ª–∫–æ—Ç–æ —á–∏—Å–ª–æ, –∫–æ–µ—Ç–æ —Å–µ –¥–µ–ª–∏ –µ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –Ω–∞ $A$ –∏ $B$.
–§–æ—Ä–º—É–ª–∞:
$$\text{lcm}(a, b) = \frac{|a \cdot b|}{\gcd(a, b)}$$

–ü—Ä–∏ –ø—Ä–æ–≥—Ä–∞–º–∏—Ä–∞–Ω–µ, –≤–∏–Ω–∞–≥–∏ –ø—ä—Ä–≤–æ –¥–µ–ª–µ—Ç–µ, –ø–æ—Å–ª–µ —É–º–Ω–æ–∂–∞–≤–∞–π—Ç–µ, –∑–∞ –¥–∞ –∏–∑–±–µ–≥–Ω–µ—Ç–µ –ø—Ä–µ–ø—ä–ª–≤–∞–Ω–µ –Ω–∞ —Ç–∏–ø–∞:
```cpp
long long lcm(long long a, long long b) {
    if (a == 0 || b == 0) return 0;
    return (a / gcd(a, b)) * b;
}
```
*   **–í–≥—Ä–∞–¥–µ–Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è**: `std::lcm(a, b)` (C++17).

## 4. –†–∞–∑—à–∏—Ä–µ–Ω –∞–ª–≥–æ—Ä–∏—Ç—ä–º –Ω–∞ –ï–≤–∫–ª–∏–¥ (Extended Euclidean Algorithm)

–û—Å–≤–µ–Ω –ù–û–î, –∞–ª–≥–æ—Ä–∏—Ç—ä–º—ä—Ç –º–æ–∂–µ –¥–∞ –Ω–∞–º–µ—Ä–∏ –∏ –∫–æ–µ—Ñ–∏—Ü–∏–µ–Ω—Ç–∏—Ç–µ $x$ –∏ $y$ –≤ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ—Ç–æ –Ω–∞ –ë–µ–∑—É (Bezout's identity):
$a \cdot x + b \cdot y = \gcd(a, b)$

–¢–æ–≤–∞ –µ –∫–ª—é—á–æ–≤–æ –∑–∞ –Ω–∞–º–∏—Ä–∞–Ω–µ –Ω–∞ **–ú–æ–¥—É–ª—è—Ä–Ω–æ –æ–±—Ä–∞—Ç–Ω–æ** —á–∏—Å–ª–æ (Modular Inverse) –ø—Ä–∏ –¥–µ–ª–µ–Ω–∏–µ –ø–æ –º–æ–¥—É–ª.

```cpp
int extended_gcd(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1; y = 0;
        return a;
    }
    int x1, y1;
    int d = extended_gcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - y1 * (a / b);
    return d;
}
```

    if (n <= 1) return false;

### 6.4. –ë—Ä–æ–π –¥–µ–ª–∏—Ç–µ–ª–∏

–ò–∑–ø–æ–ª–∑–≤–∞–π–∫–∏ —Ñ–∞–∫—Ç–æ—Ä–∏–∑–∞—Ü–∏—è: –ê–∫–æ $n = p_1^{a_1} \cdot p_2^{a_2} \cdot ... \cdot p_k^{a_k}$, —Ç–æ–≥–∞–≤–∞:

### 6.5. –°—É–º–∞ –Ω–∞ –¥–µ–ª–∏—Ç–µ–ª–∏

–§–æ—Ä–º—É–ª–∞: $\sigma(n) = \frac{p_1^{a_1+1} - 1}{p_1 - 1} \cdot \frac{p_2^{a_2+1} - 1}{p_2 - 1} \cdot ...$

```cpp
long long sumOfDivisors(int n) {
    long long sum = 1;
    
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            int exp = 0;
            while (n % i == 0) {
                exp++;
                n /= i;
            }
            // –°—É–º–∞ –Ω–∞ –≥–µ–æ–º–µ—Ç—Ä–∏—á–Ω–∞ –ø—Ä–æ–≥—Ä–µ—Å–∏—è: (p^(e+1) - 1) / (p - 1)
            sum *= (pow(i, exp + 1) - 1) / (i - 1);
        }
    }
    
    if (n > 1) sum *= (n + 1);
    
    return sum;
}
```

### 7.2. –ú–æ–¥—É–ª—è—Ä–Ω–æ —Å—Ç–µ–ø–µ–Ω—É–≤–∞–Ω–µ (Fast Exponentiation)

long long modPow(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        base = (base * base) % mod;
        exp /= 2;
    }
    
    return result;
}
```

### 8.2. –ë–∏–Ω–æ–º–Ω–∏ –∫–æ–µ—Ñ–∏—Ü–∏–µ–Ω—Ç–∏ C(n, k) = n! / (k! * (n-k)!)

## 9. –ö–∏—Ç–∞–π—Å–∫–∞ —Ç–µ–æ—Ä–µ–º–∞ –∑–∞ –æ—Å—Ç–∞—Ç—ä—Ü–∏—Ç–µ (CRT)

- ...
- $x \equiv a_k \pmod{m_k}$

–∫—ä–¥–µ—Ç–æ $m_1, m_2, ..., m_k$ —Å–∞ –≤–∑–∞–∏–º–Ω–æ –ø—Ä–æ—Å—Ç–∏.

```cpp
long long chineseRemainder(vector<int> num, vector<int> rem) {
    long long prod = 1;
    for (int i = 0; i < num.size(); i++) {
        prod *= num[i];
    }
    
    long long result = 0;
    for (int i = 0; i < num.size(); i++) {
        long long pp = prod / num[i];
        result += rem[i] * modInverse(pp, num[i]) * pp;
    }
    
    return result % prod;
}
```

## 10. –§—É–Ω–∫—Ü–∏—è –Ω–∞ –û–π–ª–µ—Ä œÜ(n)

–ë—Ä–æ–π –Ω–∞ —á–∏—Å–ª–∞—Ç–∞ –æ—Ç 1 –¥–æ n, –∫–æ–∏—Ç–æ —Å–∞ –≤–∑–∞–∏–º–Ω–æ –ø—Ä–æ—Å—Ç–∏ —Å n.

–§–æ—Ä–º—É–ª–∞: $\phi(n) = n \cdot \prod_{p|n} (1 - \frac{1}{p})$

```cpp
int phi(int n) {
    int result = n;
    
    for (int p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            while (n % p == 0) {
                n /= p;
            }
            result -= result / p;
        }
    }
    
    if (n > 1) {
        result -= result / n;
    }
    
    return result;
}
```

**–ü—Ä–∏–ª–æ–∂–µ–Ω–∏–µ**: –ú–∞–ª–∫–∞ —Ç–µ–æ—Ä–µ–º–∞ –Ω–∞ –§–µ—Ä–º–∞ –æ–±–æ–±—â–µ–Ω–∞ (–¢–µ–æ—Ä–µ–º–∞ –Ω–∞ –û–π–ª–µ—Ä):
$$a^{\phi(m)} \equiv 1 \pmod{m}$$

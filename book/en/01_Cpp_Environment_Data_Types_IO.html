Once you have a compiler and an IDE:

  2. Write the following code:

```cpp #include using namespace std;

  1. Compile the program with the command:

üî§ Program Input and Output

üì• User Input

C++ uses cin to read data from the keyboard. Example:

using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;
    cout << "You are " << age << " years old." << endl;
    return 0;
}
